# 나조토키 도우미 (謎解き 도우미)

일본어 謎解き(나조토키) 문제를 풀 때 막히지 않도록 도와주는 단일 파일 도구 모음입니다.
인터넷 없이도 동작하는 정적 HTML 한 장(`index.html`)이며, 상단 토글로 **네 가지 도구**를 오갑니다.

| 도구 | 하는 일 |
| --- | --- |
| **도도부현** | 47 도도부현을 검색·일람·지도·실루엣으로 탐색 |
| **현청소재지** | 위와 같되 기준을 현청소재지(시 이름)로 전환 |
| **규칙 그래프** | 여러 단어의 빈칸을 「같은 글자·다음 글자」 같은 규칙선으로 잇고 제약을 전파·검증 |
| **바바누키** | 여러 단어의 겹치는 글자를 지우고 남는 글자를 읽는 퍼즐 도우미 |

## 실행

브라우저로 `index.html`을 그냥 열면 됩니다(더블클릭 또는 `file://`).
로컬 서버로 띄우려면:

```bash
python3 -m http.server 8791
# → http://localhost:8791
```

선택한 도구·옵션·입력 내용은 **새로고침 후에도 유지**됩니다(localStorage).

---

## 1. 도도부현 / 현청소재지 검색

상단 토글로 **도도부현 ↔ 현청소재지** 모드를 바꾸면 아래 네 탭이 그 기준으로 동작합니다.

### 🔎 검색
- **와일드카드**: `*` = 임의 글자(0개 이상), `?` = 정확히 1글자. 와일드카드가 없으면 부분일치.
  - 예: `??` 두 글자 / `*やま` ~やま로 끝 / `か*き` か로 시작 き로 끝 / `山*` 山으로 시작하는 한자
  - 전각 `＊` / `？`도 반각 `*` / `?`와 동일하게 처리됩니다.
- **검색 대상**: 전체 / 읽기(かな) / 한자 / 로마자 (가타카나 입력도 히라가나로 정규화)
- **글자수(かな) 필터** + **정렬**: 사전순(50음, 기본) / 글자수순 / JIS순(후생노동성)
- **기본은 간단히, 「자세히 보기」로 전체**: 결과 카드는 기본적으로 한자+요미가나만 보여 한눈에 훑기 쉽고, 「자세히 보기」를 켜면 지역·글자수·로마자·현청 등 상세 정보가 펼쳐집니다.
- **「요미가나 크게」**: 한자보다 가나를 먼저 보는 경우를 위해, 카드·실루엣 타일에서 한자와 요미가나의 위치를 맞바꿔 요미가나를 크게 표시합니다(검색·전체 보기·모양 탭 공통).
- **가리기**: 카드(또는 지도·실루엣)를 클릭하면 회색으로 흐려져 후보에서 제외됩니다. 이 상태는 **모든 탭과 새로고침 후에도 유지**되고(localStorage) 모든 뷰에 함께 반영됩니다. 가린 항목이 있을 때만 헤더에 「되돌리기」가 나타납니다.

### 📋 전체 보기
47개 전부를 **글자수 그룹 / 사전순 / JIS순 / 지역별**로 묶어서 봅니다.

### 🗾 전체 지도
일본 전도에 이름을 매핑. 호버하면 상세(읽기·로마자·지역·현청)가 뜨고, 클릭하면 라벨이 고정됩니다.
「이름 표시」 끄면 퀴즈처럼 쓸 수 있고, 「지역별 색」으로 8지방 색을 입힐 수 있습니다.

### 🟩 모양 → 이름
초록색 땅덩이 실루엣 47개를 이름(한자+かな)과 함께 봅니다 — 「이 모양이 어느 현?」 대조용. 모양을 클릭하면 회색으로 가려지고 모든 탭에 반영됩니다. 정렬·그룹은 **전체 보기와 동일**(글자수 그룹 / 사전순(50음) / JIS순(후생노동성) / 지역별).
실루엣은 본섬(가장 큰 땅덩이) 기준으로 맞춰, 멀리 흩어진 작은 섬(鹿児島의 奄美 군도, 沖縄의 외곽 섬 등) 때문에 본섬이 작아지지 않도록 했습니다.

## 2. 규칙 그래프

「○글자 단어인데 둘째 글자가 같다 / 다음 글자다」처럼 **글자 사이의 관계가 힌트로 주어지는** 나조토키를 풀기 위한 도구입니다. 빈칸 단어들을 만들고, 칸과 칸을 규칙선으로 이어 제약을 시각화·전파·검증합니다.

### 만들기
단어를 두 가지 방법으로 추가합니다.
- **단어 입력**: 단어를 적고 **「+ 단어 추가」**(또는 엔터)하면 글자가 채워진 단어가 바로 만들어집니다(가타카나는 히라가나로 정규화, 1~12글자).
- **글자 수만**: 글자를 아직 모르면 **「또는 글자 수(1~12)」 → 「+ 빈 칸」** 으로 빈칸 줄만 늘립니다.

단어는 드래그 가능한 **캔버스**와 오른쪽 **단어 목록**(밑줄 빈칸)에 동시에 나타납니다. 칸/슬롯을 클릭하면 떠오르는 입력창에서 글자를 채우거나 고칠 수 있습니다.

### 규칙선 잇기
**「🔗 규칙 연결」** 을 켜고 한 칸에서 다른 칸으로 드래그하면 규칙 종류를 고를 수 있습니다. 색상 범례는 패널 상단에 표시됩니다.

| 규칙 | 의미 | 선 |
| --- | --- | --- |
| 같은 글자 | 두 칸의 글자가 같음 | 파란 실선 |
| 같은 글자 (탁점 떼기) | 탁점·반탁점을 떼면 같은 글자 (が↔か, ぱ↔は) | 청록 점선 |
| 모음 같음 | あ단/い단… 모음이 같음 | 초록 실선 |
| 자음 같음 | か행/さ행… 자음이 같음 | 보라 실선 |
| 오십음도 다음 | 오십음도에서 N칸 뒤 글자(あ→い→…) | 주황 화살표 |
| 이로하 다음 | 이로하(いろは) 순서에서 N칸 뒤 글자 | 청록 화살표 |

규칙선을 클릭하면 삭제할 수 있고, 채워 넣은 글자가 규칙과 어긋나면 **빨간 점선(규칙 위반)** 으로 경고합니다.

### 칸 제약 · 전역 옵션
- **칸 타입**(칸 우클릭 또는 길게 누르기): 일반 / **작은 글자만(小)** / **탁점만(濁)** / **반탁점만(半)**. 해당 칸엔 그 형태만 들어가도록 제한하고, 입력값도 자동 변환합니다.
- **「작은⇄큰 같은 글자」**: 켜면 `같은 글자` 규칙에서 작은 글자(ぁゃっ)와 큰 글자(あやつ)를 같은 글자로 취급합니다.
- 규칙·글자를 채우면 다른 칸으로 제약이 **전파(추론)** 되며, 추론된 글자는 흐린 기울임으로 표시됩니다.
- **「전체 삭제」** 로 모든 단어·규칙을 비웁니다.

## 3. 바바누키 (ババ抜き)

여러 단어를 늘어놓고 **겹치는 글자를 지운 뒤 남는 글자를 읽으면 답이 되는** 나조토키(통칭 ババ抜き)를 위한 도우미입니다.

- **단어 입력**: 입력창에 단어를 적고 **엔터** 또는 **「＋ 추가」**. 띄어쓰기·줄바꿈·쉼표(반각 `,` · 일본어 `、` · 전각 `，` · 반각 일본어 `､`)로 구분하면 여러 단어를 한 번에 추가합니다. (IME가 어떤 쉼표를 내든 단어가 나뉩니다.)
- **남은 글자**: 추가된 모든 단어의 글자를 한 풀로 모아 겹치는 글자를 지우고, 남는 글자를 **읽기 순서대로** 크게 보여줍니다.
- **단어별 표시**: 각 단어를 글자 타일로 펼쳐, 지워진 글자는 취소선(회색), 남는 글자는 노란색으로 강조합니다. 단어 옆 **×** 로 그 단어만 지우거나, **「목록 비우기」** 로 전체를 비웁니다.

### 겹침 규칙 (옵션)
| 규칙 | 동작 |
| --- | --- |
| **짝만 지우기** (기본) | 카드 게임 ババ抜き처럼 같은 글자를 짝지어 지움 → 개수가 홀수면 1개가 남음 |
| **겹치면 모두 지우기** | 두 번 이상 나온 글자는 전부 지움 → 딱 1번 나온 글자만 남음 |

> 「짝만 지우기」에서 같은 글자가 3개·5개처럼 홀수로 나오면, 가장 먼저 나온 1개를 남깁니다.

### 글자 맞추기 (옵션)
겹침 판정에만 적용되며 화면 표시는 입력 그대로입니다.
- **カ=か (가나 통일)**: 가타카나와 히라가나를 같은 글자로 취급(기본 켬).
- **작은⇄큰 같은 글자**: ぁ=あ, っ=つ … 작은 글자와 큰 글자를 같게.
- **탁점·반탁점 무시**: が=か, ぱ=は … 탁점·반탁점을 떼고 비교.

> 이 옵션들로 서로 다른 모양의 글자가 같은 것으로 묶여 1개만 남을 때, 화면에 보이는 글자는 **그 묶음에서 가장 먼저 입력된 형태**입니다(예: 「탁점 무시」로 `が`·`か`가 묶이면 먼저 친 쪽이 그대로 표시). 답은 실제 글자 모양이 아니라 묶인 음으로 읽으세요.

## 데이터 / 출처

- 47 도도부현 + 현청소재지: 漢字 · かな(읽기) · 로마자(헵번, 매크론 없음) · 지역(8지방) · JIS X 0401 순서(= 후생노동성 순서, 홋카이도1 ~ 오키나와47). 모든 필드 웹 교차검증 완료.
- 글자수는 **かな(히라가나) 글자 수** 기준(요음 ゃゅょ·촉음 っ 포함 각 1글자).
- 도쿄도청 소재지는 **신주쿠구(新宿区)** — 통상 「東京」으로도 표기.
- 「현청≠현명」(札幌·盛岡·仙台·水戸·宇都宮·前橋·横浜·金沢·甲府·名古屋·津·大津·神戸·松江·高松·松山·那覇 등)은 배지로 표시.
